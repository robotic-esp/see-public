3 
4 ## Description
5 
6 SEE++ is a density-based Next Best View (NBV) planning approach for obtaining 3D
7 scene observations.
8 
9 ## Installation
10 
11 - Install Dependencies
12  - sudo apt install libnlopt-dev liblemon-dev
13 - Install ROS (includes PCL)
14  - http://wiki.ros.org/melodic/Installation/Ubuntu (use full desktop install)
15 - Clone SEE into your catkin workspace
16  - cd ~/catkin_ws/src && git clone https://github.com/robotic-esp/see-public
17 - Build SEE with catkin
18  - cd ~/catkin_ws && catkin_make -DCMAKE_BUILD_TYPE=Release
19  - Using the 'Release' build enables PCL optimisations for the best performance
20 
21 ## Usage
22 
23 - Specify the desired SEE parameters (i.e., view distance (d), radius (r),
24  density (rho), frontier update limit (tau), occlusion search distance
25  (psi) and visibility search distance (ups)), the sensor parameters
26  (e.g., field-of-view, resolution and fps) and the ROS topic for sensor
27  pointclouds (sen_pts_topic) in the launch file see_core/launch/run_see.launch
28 - Run the launch file
29  - roslaunch see_core run_see.launch
30 - More information can be found on the doxygen webpage
31  - https://robotic-esp.github.io/see-public/
